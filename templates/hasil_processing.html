Preprocessing Data

📋 Alur Preprocessing (sesuai Colab)
① Teks Asli → ② Konversi Emoji → ③ Cleaning → ④ Case Folding → ⑤ Normalisasi (Emoji+Slang+Sarkasme+Data) → ⑥ Tokenisasi → ⑦ Stopword Removal → ⑧ Stemming → ⑨ TF-IDF → Naive Bayes
ℹ️ Preprocessing menggunakan library yang sama dengan Colab:
  • emoji — konversi simbol emoji ke teks deskriptif Indonesia
  • Normalisasi (1 tombol, 4 sub-langkah):
    • Emoji Teks — emoticon ASCII (:), <3, dsb.) → teks deskriptif + hapus karakter berulang
    • Kata Slang — kamus 100+ kata informal/singkatan Indonesia (gak, bgt, yg, dll.) → kata baku, didukung nlp-id Lemmatizer
    • Sarkasme — deteksi pola kalimat sarkasme (misal: "bagus banget sampai rugi") → ubah ke sentimen eksplisit
    • Data — standarisasi satuan emas, harga, format penulisan tidak standar (olshop, free ongkir, dll.)
  • nlp-id StopWord — hapus stopword Bahasa Indonesia
  • PySastrawi — stemming ke bentuk kata dasar (ECS Indonesia)
  • TF-IDF — ngram_range=(1,2), max_features=5000, sublinear_tf=True
  • Multinomial Naive Bayes — alpha=1.0, kelas: negatif / netral / positif
